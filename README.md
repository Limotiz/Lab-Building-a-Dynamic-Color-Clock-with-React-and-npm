# Personal SPA Project – Live Digital Clock

A modern, responsive React application that displays a **live digital clock** that updates automatically every second.

## How the App Works

### App.jsx

App.jsx holds the main application logic:

* Time → stores the current time
* useState → manages time state
* useEffect → updates time every second

The application automatically refreshes the displayed time without reloading the page.

---

### Time State

Stores the current date and time.

Updates every second dynamically.
